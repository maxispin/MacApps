# MacApps

Skannaa Mac-sovellukset ja lisää niihin Finder-kommentit Claude CLI:n avulla.

## Käyttö

```bash
cd MacApps
swift build
swift run
```

## Toiminta

1. Skannaa /Applications-kansion sovellukset
2. Hakee jokaiselle sovellukselle kuvauksen Claude CLI:ltä
3. Tallentaa kuvauksen Finder-kommenttiin
4. Finderissa voit etsiä sovelluksia kuvauksen perusteella
